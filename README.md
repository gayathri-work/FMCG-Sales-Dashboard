# FMCG-Sales-Dashboard
Interactive Power BI dashboard for FMCG sales analysis using Power BI, Python, and Google Colab.

## Project Overview
This project analyzes FMCG sales data using Power BI and Python. The dashboard provides insights into revenue trends, brand performance, promotional effectiveness, channel contribution, and delivery efficiency.

## Tools Used
- Power BI
- Python
- Google Colab
- Pandas
- NumPy

## Dashboard Features
- Revenue Analysis
- Monthly Sales Trends
- Top 5 Brands
- Promotion Impact Analysis
- Channel-wise Revenue Distribution
- Delivery Performance Metrics

## Project Files
- FMCG_Report.pdf
- FMCG_Sales_Dashboard.pbix
- Data_Cleaning.ipynb
